# AI Customer Assistant

企业内部 AI 客服回复助手 — 帮助员工用英语回复国外客户。

## 功能

- 输入客户英文消息，AI 生成专业回复建议
- 支持多种消息类型（咨询、投诉、反馈、技术支持、订单）
- 可选回复语气（专业、友好、正式、轻松）
- 知识库检索（RAG），基于你的 FAQ/产品手册生成更准确的回复
- 知识库管理，随时添加/更新文档
- 复制功能，一键复制 AI 生成的回复

## 使用方法

### 1. 打开网页

部署后访问：`https://g752152605-ops.github.io/ai-customer-assistant`

### 2. 输入客户消息

在左侧输入框粘贴客户发来的英文消息。

### 3. 选择设置

- **消息类型**：咨询 / 投诉 / 反馈 / 技术支持 / 订单
- **回复语气**：专业 / 友好 / 正式 / 轻松

### 4. 生成回复

点击「生成回复」，AI 将在右侧生成回复建议。

### 5. 审核并发送

AI 仅供参考，请员工审核后修改再发送。

## 知识库管理

点击右上角「知识库」按钮，可以：

- 查看已有文档数量和知识片段
- 添加新的知识库内容（FAQ、产品手册、回复模板等）
- 删除已有文档

知识库内容越多，AI 生成的回复越准确。

## 技术栈

- 前端：原生 HTML + CSS + JavaScript
- 部署：GitHub Pages
- AI：MiniMax API (MiniMax-M2.7-highspeed)
- 向量检索：客户端本地 RAG（无需额外后端）

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 安全说明

- API Key 已硬编码在前端，适用于内部小团队使用
- 知识库存储在浏览器 localStorage 中
- 不记录任何客户消息到服务器
